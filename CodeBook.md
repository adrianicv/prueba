## Human Activity Recognition Using Smartphones Tidy Dataset Code Book 


* Subject
 - Id number for the subjects.

* Activity
 - Output of the observation.It can take 6 values:
   * LAYING 
   * SITTING 
   * STANDING 
   * WALKING 
   * WALKING_DOWNSTAIRS 
   * WALKING_UPSTAIRS
